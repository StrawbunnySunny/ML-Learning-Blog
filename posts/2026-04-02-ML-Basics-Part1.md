What is it?
- Machine that learns off of historical data & creates its own rules to predict future data points (yes this can be a problem, see overfitting)
- Requires extensive data collection
- Clean data is important
- ML is a subset of AI (classified as ANI, Artifical Narrow Intelligence) 

Types of ML
1. Supervised Learning 
- This uses labelled data, which can be a time consuming process
- Labelled data is just the labelled outcome
- This means that the machine can see the outcomes and predict based on that 
2. Unsupervised Learning 
- Recognizes patterns, less focused on regression/classification
- Has no labelled data, meaning it cannot see the outcome that it is predicting
-  Example would be K-means clustering
3. Reinforcement Learning
- Akkio describes it as “carrot and stick”
- An agent is assigned and given autonomy (not given much guidance on what to do) and they’d receive penalties for negative results and rewards for positive results
- The algorithm is then able to learn what to do right and what not to do… honestly this feels a bit “abusive” 😂 → operant conditioning 
4. Deep learning
- Out of all the types, this one is the one that I am most interested in
  - They all have their own merits, but this one would be the best for forecasting
- Deep neural networks combines all three (deep supervised learning, deep unsupervised, deep reinforcement)
- 3 layers → input (info inputted), hidden (processing → math) and output (the result) 
- Other techniques are shallow compared to deep learning because the deeper layers can use different types of ML 
- Feed the raw data and the layers will stack their understanding until the model can identify certain features (unlike other types where feature engineering may be required) 

Types of Goals
1. Regression → if the variable is continuous then it will be regression
- Predicting house price, stock price, etc. (ie. finding a VALUE) 
2. Classification → determining the class that the data point belongs to
- This would be recognition, binary (yes or no)
- Examples listed in the website include: default or no default, cat or dog, etc. (think of it as recognition rather than prediction a value)
