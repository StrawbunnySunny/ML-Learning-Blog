Here's a tiny snippet of my first gradient descent implementation:

```python
def gradient_descent(x, y, lr=0.01, epochs=100):
    w, b = 0.0, 0.0
    n = len(x)
    for _ in range(epochs):
        y_pred = w * x + b
        dw = (-2/n) * sum(x * (y - y_pred))
        db = (-2/n) * sum(y - y_pred)
        w -= lr * dw
        b -= lr * db
    return w, b
```

It's not much, but seeing those loss values go down is genuinely satisfying.
